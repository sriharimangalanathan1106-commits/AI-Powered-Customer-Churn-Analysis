# AI-Powered Customer Churn Analysis

## Project Overview

This project analyzes customer churn using SQL Server, Power BI, and Python. The objective is to identify customers who are likely to leave a service and provide business insights to improve customer retention.

## Tools Used

* SQL Server
* Power BI
* Python
* Jupyter Notebook
* Random Forest Machine Learning Model

## Project Workflow

1. Data Extraction and Loading (ETL)
2. Data Cleaning and Transformation
3. Exploratory Data Analysis (EDA)
4. Power BI Dashboard Development
5. Customer Churn Prediction using Random Forest
6. Visualization of Predicted Churn Profiles

## Key Features

* SQL-based ETL and data cleaning
* Interactive Power BI dashboards
* Customer segmentation and churn analysis
* Machine Learning-based churn prediction
* Business insights for customer retention strategies

## Skills Demonstrated

SQL, SQL Server, Python, Power BI, ETL, Data Analytics, Data Visualization, Machine Learning, Random Forest
